You can open up the browser console and play around with the painter API. Suggestions:

  • painter.set_zoom(4) // Only looks good on Firefox at this point.
  • painter.set_palette_color(1, [127, 0, 255, 255])
  • painter.main_color = painter.add_palette_color([255, 0, 127,255])
  • painter.secondary_color = 1 // Right-click on canvas to use this color.

Etc.

Properties and methods starting with underscore (e.g. painter._redraw_everything()) are for internal use only.
